# create-faitjesed-generator-app

A CLI tool for generating Node.js backend projects (Fastify + TypeScript + TypeORM).

---

## 🚀 Features

- Fastify server template
- TypeScript preconfigured
- TypeORM setup ready
- Barrel imports
- Prebuilt scripts (dev / build / start)
- ESLint + Prettier ready structure
- Instant project scaffolding

---

## 📦 Installation

No installation required (recommended):

```bash
npx create-faitjesed-generator-app my-app
